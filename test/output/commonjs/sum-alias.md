<a name="module_sum-alias.sum"></a>
###\~sum
Sums two numbers together

**Params**

-  `number` - first number
-  `number` - second number

**Scope**: inner member of [sum-alias](#module_sum-alias)  
**Returns**: `number` - the result  
**Example**  
```js
var result = sum(1, 1);
```
